Integration preliminaries

A path in the complex plane is a continuous function

\[ \mathcal{C}: z(t) = x(t)+iy(t), \quad t\in[a, b], \quad a, b\in\mathbb{R} \]

that connects two points in the complex plane $z_1=z(a)$ and $z_2=z(b)$

Familiar (and important) examples include

  • straight line between the points $z_1, z_2$
    • \[ \quad\quad\quad\color{red}{\boxed{\color{white}{ \mathcal{C}: z(t) = (1-t)z_1 + tz_2, \quad t\in[0,1]}}} \]
  • The unit circle \[ \quad\quad\quad\color{red}{\boxed{\color{white}{ \mathcal{C}: \cos{t}+i\sin{t}\quad t\in[0, 2\pi)}}} \]

A closed path is one such that $z_1=z(a)=z(b)=z_2$

A simple path is one that does not self intersect

The opposite path $-\mathcal{C}$ is the path $\mathcal{C}$ traversed in the opposite direction

Fundamental theorem of contour integration


Theorem 15.1(FTCI)

If $\mathcal{C}$ is a simple path in the complex plane from $z_1$ to $z_2$ and $f$ is continuous with antiderivative $F$ then

\[ \int_\mathcal{C}f(z)\mathrm{d}z = F(z_2)-F(z_1) \] $~$

Note

  • See the online notes for a proof of this result
  • Recall that an antiderivative of $f$ is a function $F$ satisfying the relation $F'=f$

Two immediate consequences of Theorem 15.1 are

  1. If two curves $\mathcal{C_1}, \mathcal{C}_2$ start and end at the same points $z_1$ and $z_2$ then

\[ \color{red}{\boxed{\color{white}{ \int_\mathcal{C_1} f(z)\mathrm{d}z = \int_\mathcal{C_2} f(z)\mathrm{d}z = F(z_2)-F(z_1)}}} \]

  2. If a closed contour is defined by $\mathcal{C} = \mathcal{C_1}-\mathcal{C_2}$

\[ \color{red}{\boxed{\color{white}{ \int_\mathcal{C}f(z)\mathrm{d}z = \int_\mathcal{C_1} f(z)\mathrm{d}z-\int_\mathcal{C_2} f(z)\mathrm{d}z = 0}}} \]

i.e. integrals of analytic functions are path indepenent

Cauchy's integral theorem

Theorem 15.2  (Cauchy's Integral Theorem)

If $f(z)$ is analytic on and within a closed curve $\mathcal{C}$ then

\[ \oint_\mathcal{C} f(z)\mathrm{d}z = 0 \] $~$

Proof

The proof is an application of Green's theorem which states

\[ \color{#00FF00}{\boxed{\color{white}{ \oint_\mathcal{C} f_1\mathrm{d}x +f_2\mathrm{d}y = \int\int_\mathcal{D}\left(\frac{\partial f_2}{\partial x} - \frac{\partial f_1}{\partial y}\right)\mathrm{d}x\mathrm{d}y}}} \]

Here, $\mathcal{D}$ is the region enclosed by the closed curve $\mathcal{C}$

Proof (ctd)

Writing $f(z) = u(x,y)+iv(x,y)$ and $\mathrm{d}z=\mathrm{d}x+i\mathrm{d}y$ gives

\[ \begin{align*} I &= \oint_\mathcal{C}(u+iv)(\mathrm{d}x+i\mathrm{d}y)\\ &=\oint_\mathcal{C}(u\mathrm{d}x-v\mathrm{d}y) +i\oint_\mathcal{C}(v\mathrm{d}x+u\mathrm{d}y)\\ \end{align*} \]

Setting $f_1=u, f_2=-v$ in the first integral and $f_1=v, f_2=u$ in the second, we can apply Green's theorem to obtain

\[ \int\int_\mathcal{D} \left(\frac{\partial (-v)}{\partial x}-\frac{\partial u}{\partial y}\right) + i\left(\frac{\partial u}{\partial x}-\frac{\partial v}{\partial y}\right)\mathrm{d}x\mathrm{d}y \]

which proves the result since by the C-R equations we have

\[ u_x=v_y \text{ and } v_x=-u_y \implies u_x-v_y=0 \text{ and } -u_y-v_x=0 \] and so \[I = \int_\mathcal{C}f(z)\mathrm{d}z=0\]

Contour integration using a path representation

Theorem 15.3  (Integration via a path)

Let $\mathcal{C}$ be a piecwise smooth path, represented by $z(t)$, where $t\in[a,b]$. Let $f(z)$ be a continuous function on $\mathcal{C}$. Then

\[ \int_\mathcal{C}f(z)\mathrm{d}z = \int_a^b f(z(t))z'(t)\mathrm{d}t \] $~$

Proof

Note This method of integration can be applied to any continuous function regardless of whether an antiderivative exists

Example 15.1

Compute the integral of $f(z)=z^2$ along the straight line from $z_0=0$ to $z_1=1+i$

  1. using the fundamental theorem of contour integration

  2. by constructing a suitable parameterisation of the straight line segment joining the two points


Solution

Hopefully it is clear that an anti-derivative for the function $f(z)=z^2$ is given by $\displaystyle \frac{1}{3}z^3$ and so

\[ \int_0^{1+i} f(z)\mathrm{d}z = F(z_2)-F(z_1) = \frac{1}{3}(1+i)^3-\frac{1}{3}0^3 = \color{red}{\boxed{\color{white}{\frac{2}{3}(i-1)}}} \]

For part (b) we need to parameterise the straight line between $z_1=0$ and $z_2=1+i$

Using the formula from earlier we have that

\[ \begin{align*} \mathcal{C}: z(t) &= (1-t)z_1+tz_2\\ &= t(1+i) =\color{#00FF00}{\boxed{\color{white}{ t+it\quad t\in[0,1]}}} \end{align*} \]

Differentiating $z$ gives $z'(t) = 1+i$ and since $\displaystyle f(z(t)) = (t+it)^2$ we have that

\[ \int_\mathcal{C}f(z)\mathrm{d}z = \int_0^1 f(z(t))z'(t)\mathrm{d}t = \int_0^1 (t+it)^2(1+i)\mathrm{d}t \]

Multiplying out the brackets gives

\[ \begin{align*} \int_0^1 (t+it)^2(1+i)\mathrm{d}t = (1+i)\int_0^1 2it^2\mathrm{d}t &= 2i(1+i)\left[\frac{1}{3}t^3\right]_0^1\\ &=\color{red}{\boxed{\color{white}{ \frac{2}{3}(i-1)}}} \end{align*} \]

Example 15.2

By integrating $f(z)=1/z$ counterclockwise around the unit circle, show that

\[ \oint_{\mathcal{C}}\frac{\mathrm{d}z}{z} = 2\pi i \]

Solution

Note that $f(z)=1/z$ is not analytic within the region enclosed by the curve $\mathcal{C}$

It has a first order pole at $z_0=0$

To solve we need to paramterise the curve $\mathcal{C}$:

\[ \mathcal{C}: z(t) = \cos{t}+i\sin{t}\quad t\in[0, 2\pi) \]

Noting that

\[ \color{#00FF00}{\boxed{\color{white}{ z'(t) = ie^{it} \quad \text{and} \quad f(z(t)) = \frac{1}{e^{it}} = e^{-it}}}} \]

We have that

\[ \begin{align*} \oint_\mathcal{C} \frac{1}{z}\mathrm{d}z &= \int_0^{2\pi}e^{-it}\cdot ie^{it}\mathrm{d}t\\ &=i\int_0^{2\pi}\mathrm{d}t\\ &=i\Bigg[~t~\Bigg]_0^{2\pi} = \color{red}{\boxed{\color{white}{2\pi i}}} \end{align*} \]

More generally, it can be shown that

\[ \color{red}{\boxed{\color{white}{ \int_\mathcal{C}\frac{1}{z-z_0}\mathrm{d}z = 2\pi i}}} \]

for $z_0\in\mathbb{C}$

This result will prove useful moving forward

Exercise: prove the above result

Cauchy's derivative formula

Theorem 15.4  (Cauchy's derivative formula)

If $f(z)$ is analytic on and within a simlpe closed curve $\mathcal{C}$ and $z_0$ is a point within $\mathcal{C}$ then

\[ f^{(n)}(z_0) = \frac{n!}{2\pi i}\oint_\mathcal{C}\frac{f(z)}{(z-z_0)^{n+1}}\mathrm{d}z \] $~$
Note

$\bullet$ The above theorem allows us to comute integrals by rearrangment

$\bullet$ Setting $n=0$ gives Cauchy's integral formula:

\[ \color{red}{\boxed{\color{white}{ f(z_0) = \frac{1}{2\pi i}\oint_\mathcal{C} \frac{f(z)}{z-z_0}\mathrm{d}z}}} \]

Example 15.3

Use the Cauchy derivative formula to compute the integral

\[ I = \int_\mathcal{C}\frac{z^3}{(z-2)^2}\mathrm{d}z \]

where $\mathcal{C}$ is the curve $|z|=3$

Solution

Since $\displaystyle f(z) = z^3, f'(z)=3z^2, z_0=2 \text{ and } n = 1$ we have from rearrangement of Cauchy's derivative formula that

\[ \oint_\mathcal{C} \frac{z^3}{(z-2)^2}\mathrm{d}z = 2\pi i f'(z_0) = 24\pi i \]
