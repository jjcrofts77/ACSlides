Analytic functions

We can use the CR equations to determine an analytic function given either its real or imaginary part i.e. either $u$ or $v$

Example 13.1

Determine the analytic function \[f(z) = u(x,y)+iv(x,y)\] whose real part is given by \[u(x,y) = x^2-y^2\]

Solution

By the CR equations we have that

\[ u_x = v_y = 2x \quad \text{and} \quad v_x = -u_y = 2y \]

And so we have that \[ v(x,y) = \int 2x\mathrm{d}y = 2xy + f(x) \] and \[ v(x,y) = \int 2y\mathrm{d}x = 2xy + g(y) \]

Equating these two expressions gives

\[ 2xy+f(x) = 2xy + g(y) \implies f(x)=g(y)=\text{constant} \] And so \[ v(x,y) = 2xy + K \]

It follows that

\[\color{red}{\boxed{\color{white}{ f(z) = u(x,y)+iv(x,y) = x^2-y^2 + i\left(2xy+K\right)}}} \]

It is often convenient to write the complex function in terms of $z$

In the previous example it is fairly clear that

\[ x^2-y^2+i\left(2xy +K\right) = z^2+iK \]

In general this can be a bit tricky

Fortunately the following algorithm frequently comes to the rescue:


  1. Set $y=0$ in the expression for $f$

  2. Replace $x$ with $z$ in the resulting expression  

To understand why this work requires knowledge of the so-called identity theorem which lies outside of the scope of the current course

Example 13.2

Using the algorithm provided, write \[ f = x^2-y^2+i(2xy+k) \] in terms of $z$


Solution

Setting $y=0$ gives

\[ f = x^2+iK \]

which upon replacing $x$ by $z$ gives

\[ \color{red}{\boxed{\color{white}{ f(z) = z^2+iK}}} \]

Example 13.3

Using the algorithm provided, write \[ f = x(1-y)+i\left(\frac{x^2-y^2}{2}+y\right) \] in terms of $z$


Solution

Setting $y=0$ gives

\[ f = x+i\frac{x^2}{2} \]

which upon replacing $x$ by $z$ gives

\[ f(z) = z+i\frac{z^2}{2} \]

Exercise: check the above by substituting $z=x+iy$

Two more important results

  1. If $f(z)$ is analytic and $f'(z)=0$ then $f(z)$ is constant

    2. Proof Exercise (hint: use the CR equations)
  1. If $f(z)$ is analytic and any of $\mathrm{Re}(f), \mathrm{Im}(f)$ or $|f|$ are constant then $f(z)$ is constant

    2. Proof ($\mathrm{Re}(f)=$constant)

    We have that

    \[ \mathrm{Re}(f) = u(x,y) = K; \text{ clearly } u_x=u_y=0 \]

    Then, from the CR equations we have that \[ u_x=v_y=0 ~~\text{and}~~ u_y=-v_x=0 \]

    and so clearly $v$ equals a constant

    See the online notes for proofs of the remaining two results

Example 13.4

Show that the function $f(z)=z^3$ is harmonic


Solution

Recall that $f$ is harmonic if $\nabla^2u=\nabla^2v=0$

 Now \[ \begin{align*} f = z^3 &= (x+iy)^3\\ &= (x+iy)(x^2+2ixy-y^2)\\ &= x^3+2ix^2y-xy^2+iyx^2-2xy^2-iy^3\\ &= \left(x^3-3xy^2\right) + i\left(3yx^2-y^3\right) \end{align*} \] So that \[ \color{red}{\boxed{\color{white}{ u(x,y) = x^3-3xy^2 ~~\text{and}~~ v(x,y) = 3yx^2-y^3}}} \]
\[ \color{red}{\boxed{\color{white}{ u(x,y) = x^3-3xy^2 ~~\text{and}~~ v(x,y) = 3yx^2-y^3}}} \]

Since

\[ u_x = 3x^2-3y^2\quad \color{#00FF00}{\boxed{\color{white}{u_{xx} = 6x}}}\quad u_y = -6xy\quad\text{and}\quad \color{#00FF00}{\boxed{\color{white}{u_{yy} = -6x}}} \]

and

\[ v_x=6xy\quad \color{magenta}{\boxed{\color{white}{v_{xx}=6y}}}\quad v_y=3x^2-3y^2\quad\text{and}\quad \color{magenta}{\boxed{\color{white}{v_{yy}=-6y}}} \]

it is clear that

\[ \nabla^2u = u_{xx}+u_{yy}=0 ~~\text{and}~~\nabla^2v = v_{xx}+v_{yy}=0 \]

and so $f(z)=z^3$ is harmonic

Example 13.5

Express the following functions in terms of $z$:

  1. $\displaystyle f_1 = 2x+3+2iy+4i$

  2. $\displaystyle f_2 = e^{-2x}\left(\cos(2y)-i\sin(2y)\right)$

  3. $\displaystyle f_3 = x-iy$

Solution

  1. $\left(\displaystyle f_1 = 2x+3+2iy+4i\right)$

Setting $y=0$ we have that

\[ f_1 = 2x+3+4i \]

and replacing $x$ by $z$ gives $\displaystyle f_1(z) = 2z+3+4i$
  1. $\left(\displaystyle f_2 = e^{-2x}\left(\cos(2y)-i\sin(2y)\right)\right)$

Setting $y=0$ we have that

\[ f_2 = e^{-2x} \]

and replacing $x$ by $z$ gives $\displaystyle f_2(z) = e^{-2z}$

  1. $\left(\displaystyle f_3 = x-iy\right)$

Setting $y=0$ we have that

\[ f_3 = x \]

and replacing $x$ by $z$ gives $\displaystyle f_3(z) = z$

This is clearly wrong since $f_3=\bar{z}$

In fact the method breaks down for functions of the form $f(z,\bar{z})$

Example 13.6

If $u(x,y)=x(1-y)$, find $v(x,y)$


Solution

From the CR equations we know that

\[ u_x = v_y = 1-y \quad\text{and}\quad -u_y=v_x=x \]

Integrating wrt to $y$ we obtain two expressions for $v$:

\[ v(x,y) = \int 1-y \mathrm{d}y = y-\frac{y^2}{2}+g(x) \]

and

\[ v(x,y) = \int x \mathrm{d}x = \frac{x^2}{2} + h(y) \]

Equating these two expressions

\[ y-\frac{y^2}{2}+g(x) = \frac{x^2}{2} + h(y) \]

This $\implies$ that

\[ g(x) = \frac{x^2}{2}+c_1\quad \text{and} \quad h(y) = y-\frac{y^2}{2}+c_2 \]

and so

\[ v(x,y) = \frac{x^2}{2}+y-\frac{y^2}{2}+K \]

The function

\[\color{red}{\boxed{\color{white}{ \displaystyle f(z) = u(x,y)+iv(x,y) = x(1-y)+i\left(\frac{x^2}{2}+y-\frac{y^2}{2}+K\right)}}} \]

Using our algorithm we can rewrite this function in terms of $z$

\[\color{magenta}{\boxed{\color{white}{ \displaystyle f(z) = x(1-y)+i\left(\frac{x^2}{2}+y-\frac{y^2}{2}+K\right)}}} \]

Setting $y=0$ gives

\[ f = x+i\left(\frac{x^2}{2}+K\right) \]

and replacing $x$ by $z$ gives

\[\color{red}{\boxed{\color{white}{ f = z+i\left(\frac{z^2}{2}+K\right)}}} \]
