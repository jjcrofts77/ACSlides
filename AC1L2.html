Basic concepts


  • A partial differential equation (PDE) is an equation involving one or more partial derivatives of an unknown function

  • For example (here $u=u(x,t)$)
    • \[ \color{red}{\boxed{\color{white} u_{xx} = u_{tt},\quad u_{xx}=0\quad\text{and}\quad uu_t = u_x}} \]
  • The order of the highest derivative is called the $\color{yellow}{\text{order of the PDE}}$

  • As with ODEs we say the $\color{green}{\text{PDE is linear}}$ if it is of first degree in the unkown function and its derivatives

Classifying second order linear PDEs

  • Linear second order PDEs can be classified into 3 types
  • Such a PDE has the following general form:
    • \[ \color{red}{\boxed{\color{white} Au_{xx}+2Bu_{xy}+Cu_{yy}+Du_x+Eu_y+Fu=0}} \]
  • Here $A, B, C,\ldots$ are constants (in this course) and $u=u(x,t)$
  • The three cases are:
    • \[ \color{yellow}{\boxed{\color{white} \begin{align} &\text{if } AC-B^2>0 \text{ the PDE is \color{red}{\bf elliptic}}\\ &\text{if } AC-B^2=0 \text{ the PDE is \color{green}{\bf parabolic}}\\ &\text{if } AC-B^2<0 \text{ the PDE is \color{yellow}{\bf hyperbolic}}\\ \end{align} }} \]

Example 2.1 (a) Classify the following PDE \[ \color{red}{\boxed{\color{white} \frac{\partial^2u}{\partial x^2}-2\frac{\partial^2u}{\partial x\partial y}+\frac{\partial^2u}{\partial y^2}=0 }} \]

    Method of solution depend on PDE type

    For example
    • Parabolic PDEs can be transformed into a form analogous to the heat equation and so solved with standard methods

    • The famous $\color{yellow}{\text{Black Scholes PDE}}$ can be solved in this way to obtain the BS formula that is used to price financial derivatives
      • \[ \color{red}{\boxed{\color{white} \frac{\partial V}{\partial t}+\frac{1}{2}\sigma^2S^2\frac{\partial^2V}{\partial S^2}+rS\frac{\partial V}{\partial S}-rV=0}} \]
    • Hyperbolic and elliptic PDEs can similarly be transformed to a form similar to the wave and Laplace equations, respectively

    Solving PDEs like ODEs

    • Sometimes we can use methods available for solving ODEs

    • More specifically, if the PDE contains derivatives with respect to only one variable it can be treated as an ODE


    Example 2.2   Solve   $\color{red}{\boxed{\color{white} u_{xx}-u=0}}$  (here $u=u(x,y)$) as an ODE
    Solution
    Since the PDE has only derivatives in $x$ we can treat it like the ODE: \[u''-u=0\] which we solve by using the slight variation on our usual ansatz...
    ... that is we take   $\color{red}{\boxed{\color{white} u(x,y)=A(y)e^{mx}}}$  and substitute this into our equation: \[ A(y)m^2e^{mx}-A(y)e^{mx} = A(y)e^{mx}\left(m^2-1\right) =0 \] This results in two solutions \[ u_1(t)=A_1(y)e^{x} ~\text{ and }~ u_2(t)=A_2(y)e^{-x} \] We combine these linearly to obtain the general solution \[ \color{yellow}{\boxed{\color{white} u(x,y) = A_1(y)e^{x}+A_2(y)e^{-x} }} \]
    Example 2.3   Solve   $\color{red}{\boxed{\color{white} u_{xy}+u_x=0}}$  (here $u=u(x,y)$) as an ODE
    Solution
    Setting $v = u_x$ we can rewrite our equation as \[ v_y+v = 0 \implies v = A(x)e^{-y} \quad (\text{as the ODE is separable}) \] Or \[ u_x = A(x)e^{-y} \implies \color{yellow}{\boxed{\color{white} u(x,y) = B(x)e^{-y}+C(y)}} \] which is the general solution   (Here, $B(x)=\int A(x)\mathrm{d}x$)

    Solving PDEs when the solution is partly known

    Sometimes we can use the information about the system of interest (which the PDE models) to partially 'guess' the solution
    Example 2.4   Solve \[\color{red}{\boxed{\color{white} z+\frac{\partial^2 z}{\partial x^2}+\frac{\partial^2z}{\partial t^2}-2\frac{\partial z}{\partial t}=0}}\] (Here $z=z(x,t)$) subject to the conditions \[ z(x,0)=4\sin(3x)\quad\text{and}\quad z(0,t)=0 \] given that   $\color{green}{\boxed{\color{white} z(x,t) = f(x)e^{-2t}}}$
    Solution
    We start by working out the necessary derivatives \[ \begin{align*} z_t &= -2f(x)e^{-2t} \quad z_{tt} = 4f(x)e^{-2t}\\\\ z_x&=f'e^{-2t} \quad z_{xx} =f''(x)e^{-2t}\\ \end{align*} \] which upon substitution into our PDE gives \[ fe^{-2t}+f''e^{-2t}+4fe^{-2t}+4fe^{2t} = e^{-2t}\left(f''+9f\right)= 0 \] Or \[ \color{green}{\boxed{\color{white} f''+9f = 0}} \quad \left(\text{An ODE for the unknown function } ~f(x)\right) \]
    The auxillary equation is \[ m^2+9=0\implies m = \pm 3i \] And so $f$ is given by \[ f(x) = A\cos(3x)+B\sin(3x) \] Substituting this back in to the original equation for $z(x,t)$ gives \[ \color{green}{\boxed{\color{white} z(x,t) = \left(A\cos(3x)+B\sin(3x)\right)e^{-2t}}} \] as our general solution
    To obtain the particular solution...
    ... we incorporate our boundary and initial data: \[ z(0,t) = Ae^{-2t} = 0 \implies A = 0 \] And \[ \begin{align*} z(x,0) &= B\sin(3x) = 4\sin(3x)\\ \implies& B = 4 \end{align*} \] Giving the particular solution \[ \color{yellow}{\boxed{\color{white} z(x,t) = 4\sin(3x)e^{-2t}}} \]
    Example 2.5   Assuming a solution of the form $\displaystyle u(x,y)=y^\alpha f(x)$, where $\alpha$ is a constant, solve the following PDE \[ \color{red}{\boxed{\color{white} y^2u_{yy}+2yu_y-2u=0}} \] Solution
    To start \[ u_y =\alpha y^{\alpha-1}f(x) \quad \text{and} \quad u_{yy} = \alpha(\alpha-1)y^{\alpha-2}f(x) \] which upon substitution gives \[ \alpha(\alpha-1)y^{\alpha-2}f(x)+\frac{2}{y}\alpha y^{\alpha-1}f(x)-\frac{2}{y^2}y^\alpha f(x)=0 \] Rearranging gives...
    \[ \color{green}{\boxed{\color{white} y^{\alpha-2}f(x)\left[\alpha^2+\alpha-2\right] = 0 }} \] This equation has solutions (s.t $f(x)$ is non-zero) provided \[ \alpha^2+\alpha-2 = (\alpha-1)(\alpha+2)=0 \] Thus we have two solutions: \[ u_1(x,y) = yf_1(x)\quad\text{and}\quad u_2(x,y) = y^{-2}f_2(x) \] Which we combine linearly to obtain the general solution \[ \color{yellow}{\boxed{\color{white} u(x,y) = Ayf_1(x)+\frac{B}{y^2}f^2(x)}} \]
