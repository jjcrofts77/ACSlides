Residues

From Cauchy's theorem we know that if $f$ is analytic on and within a closed curve containing the point $z_0$ then

\[ I = \oint_\mathcal{C} f(z)\mathrm{d}z = 0 \]

In general, if $f(z)$ is not analytic within $\mathcal{C}$ then $I$ is nonzero

If $z_0$ is the only singularity in $\mathcal{C}$ then the integral can be easily evaluated

To see this let us represent $f(z)$ by its Laurent series about the point $z_0$: \[ f(z) = \sum_{n=0}^\infty a_n(z-z_0)^n + \sum_{n=1}^{\infty} \frac{b_{n}}{(z-z_0)^n} \]

Intergrating term by term gives...

\[ \oint_\mathcal{C}f(z)\mathrm{d}z = \sum_{n=0}^\infty \oint_\mathcal{C}a_n(z-z_0)^n\mathrm{d}z + \sum_{n=1}^\infty \frac{b_n}{(z-z_0)^n}\mathrm{d}z \]

Here $\mathcal{C}$ is the curve given by $|z-z_0|=r$

Now recall that

\[ \oint_\mathcal{C}(z-z_0)^n\mathrm{d}z = \begin{cases} 2\pi i,&n=-1\\0,&\text{otherwise}\end{cases} \]

So that the only non-zero term in the above integral is the $b_1$ term so that

\[ \color{red}{\boxed{\color{white}{ \oint_\mathcal{C}f(z)\mathrm{d}z = 2\pi ib_1}}} \]

The $b_1$ term is the coefficient on the first negative power term in the LS

The ideas presented above can be extended using the multiple annulus theorem

If $\mathcal{C}_i$ are $m$ closed paths within $\mathcal{C}$ each enclosing an isolated singularity $z_i$ of $f(z)$ then it can be shown that

\[ \oint_\mathcal{C}f(z)\mathrm{d}z = 2\pi i\sum_{n=1}^m b_{1}^{(n)} \]

where $b_1^{(n)}$ indicates the coefficient associated with the first negative power term in the LS of $f$ expanded about the point $z_i$

Definition 19.1  (Residue)

The non-zero part of the integral arising due to the singularity at $z_0$ is known as the residue of $f(z)$ and is denoted $\mathrm{Res}(f;z_0), $ i.e.

\[ \oint_\mathcal{C} f(z)\mathrm{d}z = 2\pi i\mathrm{Res}(f;z_0) \]

The Residue Theorem

By simple analogy we obtain the following important result

Theorem 19.1  (Residue theorem)

If $f(z)$ is analytic on and inside the simple closed path $\mathcal{C}$ except at a finite number of isolated singularities of $f$, $z_1, z_2, \ldots, z_m$, then

\[ \oint_\mathcal{C} f(z)\mathrm{d}z = 2\pi i\sum_{n=1}^m \mathrm{Res}(f;z_n) \]

Example 19.1

Find the singularities and corresponding residues of

\[ f(z) = \frac{z^2+5z+3}{(z-1)(z+2)^2} \]

Solution

The function has 2 poles:

\[ z_1=1 \quad\text{and}\quad z_2=-2 \]

which are first and second order, respectively

Let us rewrite $f$ using partial fractions

\[ f(z) = \frac{z^2+5z+3}{(z-1)(z+2)^2} = \frac{1}{z-1}+\frac{1}{(z+2)^2} \]

(a) $z_1=1$

Since we are expanding about $z_1=1$, we want to write $f$ in the form

\[ \begin{align*} f(z) &= \frac{1}{z-1}+\frac{1}{(z+2)^2}\\ &=\sum_{n=0}^\infty a_n(z-1)^n + \sum_{n=1}^\infty \frac{b_n}{(z-1)^n} \end{align*} \]

The first term in line one above is already in the correct form

The second term is analytic at $z=1$ so we can write it as a Taylor series:

\[ \frac{1}{(z+2)^2} = \frac{1}{9}-\frac{2}{27}(z-1)+\frac{1}{27}(z-1)^2-\cdots \quad\text{\textcolor{red}{(exercise)}} \]

Thus the Laurent series at $z_1=1$ is

\[ f(z) = \color{red}{\boxed{\color{white}{\frac{1}{z-1}}}}\color{white}{+\frac{1}{9}-\frac{2}{27}(z-1)+\frac{1}{27}(z-1)^2-\cdots} \implies \color{red}{\boxed{\color{white}{\mathrm{Res}(f; 1)=1}}} \]

(b) $z_2=-2$

Since we are expanding about $z_2=-2$, we want to write $f$ in the form

\[ \begin{align*} f(z) &= \frac{1}{z-1}+\frac{1}{(z+2)^2}\\ &=\sum_{n=0}^\infty a_n(z+2)^n + \sum_{n=1}^\infty \frac{b_n}{(z+2)^n} \end{align*} \]

The second term in line one above is already in the correct form

The first term is analytic at $z_2=-2$ so we can write it as a Taylor series:

\[ \frac{1}{(z-1)} = -\frac{1}{3}-\frac{z+2}{9}-\frac{1}{27}(z+2)^2-\frac{1}{81}(z+2)^3-\cdots \quad\text{\textcolor{red}{(exercise)}} \]

Thus the Laurent series at $z_2=-2$ is

\[ f(z) = \frac{1}{(z+2)^2}-\frac{1}{3}-\frac{z+2}{9}-\frac{1}{27}(z+2)^2-\frac{1}{81}(z+2)^3-\cdots \implies \color{red}{\boxed{\color{white}{\mathrm{Res}(f; -2)=0}}} \]

Additional results

Sometimes we can compute the residue without reference to the LS

1. If $z_i$ is a removable singularity then $\mathrm{Res}(f;z_i)=0$

2. If $z_i$ is a simple pole then

\[ \color{red}{\boxed{\color{white}{ \mathrm{Res}(f;z_i) = b_1 = \lim_{z\to z_i}\left\{(z-z_i)f(z)\right\}}}} \]

3. If $z_i$ is a pole of order $m$ then

\[ \color{red}{\boxed{\color{white}{ \mathrm{Res}(f;z_i) =\lim_{z\to z_i}\left\{\frac{1}{(m-1)!}\frac{\mathrm{d}^{m-1}}{\mathrm{d}z^{m-1}}\left((z-z_i)^mf(z)\right)\right\}}}} \]

If $z_i$ is an essential singularity then we need to compute the LS to determine the residue

Another result that is sometimes useful is the following:

Assuming that

\[ f(z) = \frac{p(z)}{q(z)} \quad\text{with } p(z_i)\neq 0 \]

and that $q(z)$ has a simple zero at $z_i$ then

\[ \color{red}{\boxed{\color{white}{ \mathrm{Res}(f; z_i) = \frac{p(z_i)}{q'(z_i)}}}} \]

In the above $p(z)$ and $q(z)$ are polynomial functions

Example 19.2

Using the results over page compute the residues of the function

\[ f(z) = \frac{z^2+5z+3}{(z-1)(z+2)^2} \]

thus verifying our earlier results

Solution

Since $z_1=1$ is a simple pole we have that

\[ \begin{align*} \mathrm{Res}(f; 1) &= \lim_{z\to 1}\left\{(z-1)f(z)\right\}\\ &=\lim_{z\to 1}\left\{\frac{z^2+5z+3}{(z+2)^2}\right\}= \left.\frac{z^2+5z+3}{(z+2)^2}\right|_{z=1} = \color{red}{\boxed{\color{white}{1}}} \end{align*} \]

The last line follows since $f$ is continuous

Since $z_2=-2$ is a second order pole we have that

\[ \begin{align*} \mathrm{Res}(f; -2) &= \lim_{z\to -2}\left\{\frac{\mathrm{d}}{\mathrm{d}z}\left((z+2)^2f(z)\right)\right\}\\ &=\lim_{z\to -2}\left\{\frac{\mathrm{d}}{\mathrm{d}z}\left(\frac{z^2+5z+3}{z-1}\right)\right\}\\ &=\lim_{z\to -2}\left\{\frac{z^2-2z-8}{(z-1)^2}\right\} = \left.\frac{z^2-2z-8}{(z-1)^2}\right|_{z=-2} = \color{red}{\boxed{\color{white}{0}}} \end{align*} \]

Again the last line follows since $f$ is continuous

Which is in agreement with our earlier result
