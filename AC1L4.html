Fourier series of periodic functions

Recall from last weeks lecture that given a $2\pi$ periodic function $f(x)$ we can write it as a Fourier series

\[ \color{red}{\boxed{\color{white}{ f(x) \sim \frac{a_0}{2}+\sum_{n=1}^\infty \left(a_n\cos(nx)+b_n\sin(nx)\right) }}} \]

Here $a_n$ and $b_n$ are given by the Euler integrals

\[ \color{#00FF00}{\boxed{\color{white}{ a_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos(nx)\mathrm{d}x~ \text{ and }~ b_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\sin(nx)\mathrm{d}x }}} \]

Recap:

recall that we found the Fourier series for the function

\[ f(x) = \begin{cases}1&x\in[0,\pi)\\-1 & x\in[-\pi,0)\end{cases} \]

such that $\displaystyle f(x)=f(x+2\pi)$ outside of $(-\pi,\pi)$

The FS was given by

$ \color{red}{\boxed{\color{white}{ f(x)\sim \sum_{n=1}^\infty \frac{2}{n\pi}\left( 1- (-1)^n\right)\sin(nx) }}} $

Note

  • We use $\sim$ as the FS and function differ at zero
    and ...
  • $f$ consists of sines only...
    •

    The above plot shows the partial sums (for $n=5, 10$ and $100$ terms) for the FS computed over page

    Interestingly, we can use the result

    \[\color{red}{\boxed{\color{white}{f(x) \sim\sum_{n=1}^\infty \frac{2}{n\pi}\left(1-(-1)^n\right)\sin(nx)}}}\]

    to get an approximation to $\pi$

    Evaluating $f$ at $\pi/2$ gives

    \[ f\left(\frac{\pi}{2}\right) = 1 = \frac{2}{\pi}\left(2-\frac{2}{3}+\frac{2}{5}-\cdots\right) \]

    so that

    \[ \pi = 4-\frac{4}{3}+\frac{4}{5}-\frac{4}{7}\cdots \]

    The above plot shows our approximation to $\pi$ for increasing $n$

    Example 4.1

    Compute the Fourier series of the function

    \[ f(x) = x^2,\quad x\in(-\pi, \pi) \]

    such that $f(x) = f(x+2\pi)$ outside $(-\pi,\pi)$

    Solution

    Firstly, note that since the function $f$ is even

    \[ b_n = 0 \]

    This follows since $f(x)\sin(nx)$ is odd for $f$ even

    Now

    \[ \begin{align*} a_0 = \frac{1}{\pi}\int_{-\pi}^\pi x^2\mathrm{d}x &= \frac{2}{\pi}\int_0^\pi x^2\mathrm{d}x\\ &=\frac{2}{\pi}\bigg[\frac{1}{3}x^3\bigg]_0^\pi = \color{red}{\boxed{\color{white}{\frac{2}{3}\pi^2 }}} \end{align*} \]

    To compute $a_n$

    \[ a_n = \frac{1}{\pi}\int_{-\pi}^\pi x^2\cos(nx)\mathrm{d}x = \frac{2}{\pi}\int_0^\pi x^2\cos(nx)\mathrm{d}x \]

    So that

    \[ \begin{align*} a_n &= \frac{2}{\pi}\left(\bigg[\frac{1}{n}x^2\sin(nx)\bigg]_0^\pi - \frac{2}{n}\int_0^\pi x\sin(nx)\mathrm{d}x\right)\\ &=-\frac{4}{n\pi}\left(\bigg[-\frac{1}{n}x\cos(nx)\bigg]_0^\pi+\frac{1}{n}\int_0^\pi\cos(nx)\mathrm{d}x\right)\\ &=\color{red}{\boxed{\color{white}{\frac{4}{n^2}(-1)^n}}} \end{align*} \]

    Putting the above together gives us our FS representation

    \[ \color{#00FF00}{\boxed{\color{white}{ \begin{align*} f(x) &\sim \frac{a_0}{2}+\sum_{n=1}^\infty a_n\cos(nx)+b_n\sin(nx)\\ &= \frac{\pi^2}{3} + \sum_{n=1}^\infty \frac{4(-1)^n}{n^2}\cos(nx) \end{align*} }}} \]

    Fourier sine series

    As the previous examples have alluded, knowledge of whether a function is odd or even can greatly simplify the Fourier series computations

    Let $f$ be $2\pi$ periodic.

    If $f$ is odd then its FS is given by

    \[ \color{red}{\boxed{\color{white}{ f(x)\sim \sum_{n=1}^\infty b_n\sin(nx) }}} \]

    where

    \[ b_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\sin(nx)\mathrm{d}x = \frac{2}{\pi}\int_0^\pi f(x)\sin(nx)\mathrm{d}x \]

    Fourier cosine series

    As the previous examples have alluded, knowledge of whether a function is odd or even can greatly simplify the Fourier series computations

    Let $f$ be $2\pi$ periodic.

    If $f$ is even then its FS is given by

    \[ \color{red}{\boxed{\color{white}{ f(x)\sim \frac{a_0}{2}+\sum_{n=1}^\infty a_n\cos(nx) }}} \]

    where

    \[ a_n = \frac{1}{\pi}\int_{-\pi}^\pi f(x)\cos(nx)\mathrm{d}x = \frac{2}{\pi}\int_0^\pi f(x)\cos(nx)\mathrm{d}x \]

    Example 4.2

    Compute the Fourier series of the function

    \[ f(x) = x|x|-x^3 \]

    such that $f(x)=f(x+2\pi)$ outside of $(-\pi, \pi)$

    Solution

    Since $f$ is an odd function we have $a_0=a_n = 0~~\forall n\in\mathbb{N}$

    To compute $b_n$ let us note a couple of simplifying points:

  • $f(x)\sin(nx)$ is even so we can integrate from $0$ to $\pi$ and double the result

  • On the interval $(0, \pi)$ the function \[f(x) = x|x|-x^3= x^2-x^3\]
    •

    So

    \[ \color{#00FF00}{\boxed{\color{white} { b_n =\frac{2}{\pi}\int_0^\pi \left(x^2-x^3\right)\sin(nx)\mathrm{d}x }}} \]

    We compute this integral as follows

    \[ \begin{align*} b_n &= \frac{2}{\pi}\int_0^\pi\left(x^2-x^3\right)\sin(nx)\mathrm{d}x\\ &=\frac{2}{\pi}\bigg[-\frac{1}{n}(x^2-x^3)\cos(nx)\bigg]_0^\pi\\&+\frac{2}{n\pi}\int_0^\pi(2x-3x^2)\cos(nx)\mathrm{d}x\\ &=-\frac{2}{n\pi}(\pi^2-\pi^3)(-1)^n+\frac{2}{n\pi}\bigg[\frac{1}{n}(2x-3x^2)\sin(nx)\bigg]_0^\pi\\ &-\frac{2}{n^2\pi}\int_0^\pi (2-6x)\sin(nx)\mathrm{d}x = \cdots \end{align*} \]

    ...so

    \[ \begin{align*} b_n &= -\frac{2}{n\pi}(\pi^2-\pi^3)(-1)^n-\frac{2}{n^2\pi}\int_0^\pi (2-6x)\sin(nx)\mathrm{d}x\\ &=-\frac{2}{n\pi}(\pi^2-\pi^3)(-1)^n-\frac{2}{n^2\pi}\bigg[-\frac{1}{n}(2-6x)\cos(nx)\bigg]_0^\pi\\ &-\frac{12}{n^3\pi}\int_0^\pi\cos(nx)\mathrm{d}x\\ &=\color{red}{\boxed{\color{white}{\frac{2}{n\pi}(\pi^3-\pi^2)(-1)^n+\frac{2}{n^3\pi}\left((2-6\pi)(-1)^n-2\right)}}} \end{align*} \]

    It follows that the solution is given by

    \[ \color{#00FF00}{\boxed{\color{white}{ \begin{align*} f(x)&\sim \sum_{n=1}^\infty\left[\frac{2}{n\pi}(\pi^3-\pi^2)(-1)^n\right.\\&\left.+\frac{2}{n^3\pi}\left((2-6\pi)(-1)^n-2\right)\right]\sin(nx) \end{align*} }}} \]

    Hopefully, you can see how much more work would have been involved in this question had we not noticed the function is odd
